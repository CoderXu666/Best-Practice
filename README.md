### 1、项目介绍

欢迎访问Bingo社交开源项目，这是一款功能丰富、新颖、针对年轻人交友的平台系统。 项目整体采用微服务架构，技术栈全面、主流、前沿。
感谢您的Star，期待您的加入我们，一起Fork！

### 2、模块划分

#### 公共模块【bingo-common】

用于封装项目通用所需的相关实体类、工具类、配置类、远程调用、注解、枚举、拦截器等.....

#### 用户管理【bingo-user】

该模块用于用户基本信息，例如：用户管理、礼物相关、会员权益、账户充值、Bingo Show等.....
